## 🚀 使用指南

### **方式一：使用新架构（推荐）**

1. **初始化数据库**
   ```cmd
   cd database
   python init_db.py
   cd ..
   ```

2. **启动服务**
   ```cmd
   start_all_mcps.bat
   ```

3. **验证服务**
   - 访问 http://localhost:8081/health （油井基础数据 MCP）
   - 访问 http://localhost:8082/health （日报系统 MCP）

### **方式二：使用旧版本（备用）**

**启动真实数据服务器：**
```cmd
start_true_server.bat
```

**启动模拟数据服务器：**
```cmd
start_mock_server.bat
```
