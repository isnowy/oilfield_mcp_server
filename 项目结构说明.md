## 🎯 文件分类

### **✅ 主要使用（新架构）**
- `oilfield_wells_mcp.py` - 油井基础数据 MCP 服务器
- `oilfield_dailyreports_mcp.py` - 日报系统 MCP 服务器
- `start_all_mcps.bat/ps1` - 启动新 MCP 服务器
- `common/` - 共享模块
- `README_新架构.md` - 快速开始指南

### **📦 数据库相关**
- `database/` - 所有数据库文件和脚本

### **📚 文档**
- `md/` - 所有项目文档

### **🗂️ 旧版本（保留备份）**
- `oilfield_mcp_true_server.py` - 旧版真实数据服务器
- `oilfield_mcp_server.py` - 旧版模拟数据服务器
- `start_true_server.bat/ps1` - 启动旧版真实数据服务器
- `start_mock_server.bat/ps1` - 启动旧版模拟数据服务器

### **❌ 已废弃**
- `oilfield_mcp_http_server.py` - HTTP 版本（已集成到新版）

## 🚀 快速开始

### **推荐：使用新架构**
```cmd
REM 1. 安装依赖
install_requirements.bat

REM 2. 初始化数据库（首次使用）
cd database
python init_db.py
cd ..

REM 3. 启动所有 MCP 服务器
start_all_mcps.bat
```

### **备选：使用旧版本**
```cmd
REM 启动旧版真实数据服务器
start_true_server.bat

REM 或启动旧版模拟数据服务器
start_mock_server.bat
```

## 🔧 配置文件

### **环境变量配置**
```bash
DB_HOST=localhost       # 数据库主机
DB_PORT=5432           # 数据库端口
DB_NAME=rag            # 数据库名称
DB_USER=postgres       # 数据库用户
DB_PASSWORD=postgres   # 数据库密码
DEV_MODE=true          # 开发模式（跳过权限检查）
```

参考 `librechat_mcp_config.json` 配置文件。

## 🆕 新功能亮点

1. **模块化架构** - 代码拆分成独立的 MCP 服务器
2. **共享模块** - 数据库、权限、工具函数复用
3. **更好的维护性** - 单个文件代码量减少 67%
4. **独立部署** - 每个 MCP 可以独立启动和更新
5. **故障隔离** - 单个服务故障不影响其他服务
